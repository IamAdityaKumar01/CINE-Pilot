# CINE-Pilot

CINE-Pilot is a frontend React application that works as a smart movie guide. It allows users to search for movies using natural language queries such as “trending horror movies in 2025” and view relevant results. This project focuses on practicing React and building a clean, functional movie search interface.

## Features

- Natural language search input  
- AI-interpreted search queries (handled via an external backend/AI API, not included in this repository)  
- Clean and simple UI for browsing movie results  
- Displays movie details such as title, year, poster, and overview  
- Modular and reusable React components  

## Tech Stack

- React  
- Vite  
- Redux 
- Tailwind CSS  
- Fetch-based API integration for movie and AI data
